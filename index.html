Word Meaning image
Software Set of programs and routines that allow the computer to perform certain tasks.
Code They are the instructions that a programmer writes in a programming language. Code is the stage where a programmer can read and modify a computer program.
Framework A framework provides a standard way of building applications. It consists of many predefined solutions for common functionality that are used to help build software applications faster and more standardized.
Library A library is a collection of prebuilt functions to perform specific tasks. For example, writing code that exports a database to excel would take a programmer quite a bit of time. Therefore, it can be supported by libraries that already have this function already developed.
Bug A bug is an error in the code of a program.
Domain Basically it is the text that a person enters in a browser to enter a specific website.
Hosting It is the service that provides storage of information, images, video, or any content accessible via the Web.
Java A compiled object-oriented programming language, similar in syntax to C++.
Database Base of information stored and used by a program. Traditional databases are organized by fields, records, and files.
User Interface It is the visual part of a program. It is what the user sees, and with what he interacts.
Cookies Cookies allow websites to track the actions of their users. In this way, web developers can know if a user has previously visited a site and in which browser.
identifier Name of a variable, method, class, interface, or package.
Main Class A class that contains a main method.
Network Infrastructure that allows computers to communicate one with another.
Packages Set of programs and routines that allow the computer to perform certain tasks.
tag An HTML instruction that indicates a web browser how to view a document.
UML Unified modeling language that provides notation visual standard for documenting the analysis and design of object oriented systems.
Unicode An international character encoding system managed by the Unicode consortium, Java supports Unicode.
Comments They can appear anywhere in the program, as long as they are located between the delimiters /* comment */. Comments are useful to identify the main elements of a program or to explain the underlying logic of these.
Flowchart It is the graphical representation of an algorithm. It can also be said that it is detailed representation in graphical form of how the steps in the program must be carried out in order to produce results.
